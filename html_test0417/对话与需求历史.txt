## 2. 多源数据与外部平台集成架构

### 2.1 行情数据层（Data Layer）
**接入平台清单**：
- **主要数据源**：
  - 同花顺iFinD Open API（A股实时行情、财务数据、Level-2资金流向）

- **辅助验证源**：
  - 东方财富Choice数据（基金持仓关联）
  - 新浪财经API（实时行情JSONP接口，适合前端直接拉取）
  - 腾讯财经API（实时行情，备用数据源）

**数据类型覆盖**：
- 实时行情（延迟15分钟或实时，视授权而定）
- 历史K线（日K/周K/月K，前复权/后复权）
- 财务数据（季报/年报，关键指标TTM）
- 市场情绪（涨跌家数比、涨停跌停比、北向资金流向）

- **快讯类**：财联社API（电报式快讯，24小时政策/公告）、华尔街见闻（全球宏观）

- **官方源**：上交所/深交所公告（IPO/重组/减持公告RSS）、巨潮资讯网（法定信息披露）
- **国际对标**：Bloomberg API（港股/美股映射）、TradingView（全球技术指标社区）

**内容处理逻辑**：

- 关键词标签提取（概念关联：ChatGPT→算力→CPO→中际旭创）
- 去重与相关性排序（避免重复报道刷屏）

### 2.3 分析工具集成（Tool Integration）

## 3. 全栈技术架构方案（Production-Ready）

### 3.1 前端技术栈（Frontend）
**核心框架**：Next.js 14 (App Router) + TypeScript
- **优势**：SSR利于SEO（资讯内容被搜索引擎收录）、Server Components减少客户端JS、Vercel一键部署
- **状态管理**：Zustand（轻量级全局状态）+ React Query（TanStack Query，服务端状态缓存）
- **UI组件库**：
  - Ant Design 5.0（后台管理系统风格，适合数据展示）
  - Tailwind CSS（原子化样式，自定义金融主题色）
  - Framer Motion（动画库，股价跳动、K线绘制动画）

**可视化图表**：
- **轻量级**：Lightweight Charts（TradingView出品，开源免费，适合分时图/K线）
- **进阶分析**：ECharts 5（指标叠加、多图联动、成交量柱状图）
- **交互式教学**：D3.js（自定义K线形态教学动画、资金流向桑基图）

### 3.2 后端技术栈（Backend）
**核心框架**：Node.js + NestJS（或Python FastAPI备选）
- **API设计**：GraphQL（Apollo Server）+ RESTful Hybrid（复杂查询用GraphQL，简单请求用REST）
- **实时通信**：Socket.io（WebSocket封装，实时推送模拟交易价格、资讯快讯）
- **任务队列**：BullMQ（Redis-based，处理耗时任务：批量历史数据下载、财报PDF解析）

**数据层**：
- **主数据库**：PostgreSQL 15（关系型数据：用户账户、模拟交易记录、自选股列表）
  - TimescaleDB插件（时序数据扩展，高效存储股价历史数据）
- **缓存层**：Redis 7（实时行情缓存、热点资讯、用户会话、排行榜）
- **搜索引擎**：Elasticsearch（全文搜索股票名称/代码/资讯内容，拼音模糊匹配）

### 3.3 数据采集与处理（Data Pipeline）
**爬虫/采集服务**（Python Celery分布式任务）：
- Scrapy框架（爬取东方财富研报、雪球热帖，遵守robots.txt）
- AKShare/Tushare SDK（标准化API调用，异常自动重试）
- 数据清洗：Pandas（处理除权除息数据、财务数据标准化）

**消息队列**：Apache Kafka（或RabbitMQ）
- 处理高并发行情数据推送（开盘时段数据洪峰）
- 解耦数据生产（爬虫）与消费（前端推送）

### 3.4 DevOps与部署
**基础设施**：
- **容器化**：Docker + Docker Compose（开发环境一致性）
- **CI/CD**：GitHub Actions（自动化测试、构建、部署）
- **云服务**：阿里云/腾讯云（国内低延迟）
  - ECS（部署NestJS后端）
  - RDS（托管PostgreSQL）
  - OSS（存储用户头像、教学视频、财报PDF）
  - CDN（加速静态资源、行情数据边缘缓存）

**监控与日志**：
- Sentry（前端错误监控）
- Prometheus + Grafana（后端性能监控、API响应时间、数据库慢查询）
- ELK Stack（日志收集与分析）

### 3.5 AI能力集成（大模型接口）
**LLM服务**：
- **国产合规**：百度文心一言API / 阿里通义千问 / 智谱GLM-4（金融垂类调优）
- **分析场景**：输入股票代码→自动调取实时数据→生成技术面+基本面简报（结构化JSON输出）
- **RAG增强**：LangChain + 向量数据库Pinestore/Milvus（导入已上市的投教知识库，确保回答准确）

## 4. 功能模块详细设计

### 4.1 智能学习路径（Gamification）
- **等级体系**：L1菜鸟→L2见习→L3操盘手...→L10股神，每级解锁新功能（模拟交易额度提升、高级指标查看权限）
- **成就徽章**："首单盈利"、"连续签到7天"、"MACD大师"等NFT式勋章
- **模拟盘大赛**：周赛/月赛排行榜，收益率TOP10展示（虚拟奖励）

### 4.2 计算器工具箱
- **盈亏计算器**：支持A股/港股通/可转债，自动计算不同券商佣金差异
- **定投计算器**：定投收益曲线图，对比一次性投入vs定投
- **打新收益计算器**：根据历史数据预测新股/新债中签收益
- **仓位计算器**：凯利公式自动计算最佳仓位比例

### 4.3 AI股票分析师（合规版）
**输入**：股票代码/名称 + 分析维度选择（短线技术/长线价值/事件驱动）
**处理流程**：
1. 调取实时行情数据（价格、成交量、近期龙虎榜）
2. 调取近期关联资讯（NLP情感分析）
3. LLM生成报告结构：
   - 技术面评分（1-10分）：趋势、支撑压力、量价配合
   - 基本面摘要：最新季报亮点、同行业对比
   - 风险提示：大股东减持、解禁日期、业绩预减
   - 教学要点：当前形态对应的经典技术分析案例
**输出限制**：禁止给出"买入/卖出"明确指令，仅提供"观察角度"和"学习要点"

## 5. 合规与安全架构

### 5.1 内容安全
- **敏感词过滤**：接入阿里云内容安全（绿网），用户评论/交易笔记自动审核
- **反爬虫机制**：Nginx限流（Rate Limiting），用户行为验证码（极验）
- **数据脱敏**：模拟交易数据与真实账户完全隔离，避免用户误操作

### 5.2 投资者适当性管理
- **风险测评问卷**：开户前强制完成10道题（投资经验/收入/风险偏好），评级C1-C5
- **适当性匹配**：高风险教学内容（如融资融券、期权）需C4以上用户解锁
- **冷静期提醒**：模拟交易连续亏损20%时弹出"学习提醒"，引导回归基础知识
